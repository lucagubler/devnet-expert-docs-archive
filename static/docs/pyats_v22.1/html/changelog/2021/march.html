Known Issues¶

On MacOs we have observed issues with the requests package used for REST API calls, the workaround is to set the environment variable no_proxy to a value, e.g. domain.com. Without this, we have seen crashes that halt the script execution. This issue seems to be related to a bug in MacOS python https://bugs.python.org/issue31818
