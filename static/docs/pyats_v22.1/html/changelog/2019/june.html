Changes¶

  • fixed a bug with regex for file listing match in FileUtils ftp plugin

  • fixed a bug with Easypy job file loader where it does not add job directory to sys.path.

  • updated SMTP server customization documentation to reflect latest uses of pyATS configuration file

  • [Cisco Internal only] fixed a bug in pyats.cisco TIMS uploading module where archive_file is not uploaded as an attribute field
