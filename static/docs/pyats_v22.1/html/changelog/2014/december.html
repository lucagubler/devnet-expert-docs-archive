User Impacting ChangesÂ¶

  • banner API is now part of ats.atslog.utils and no longer performs the actual logging. It returns a pre-formatted text to be logged instead.

    import logging

from ats.atslog.utils import banner

logger = logging.getLogger()

# printing a banner
logger.info(banner('this is a banner text'))

  • job file now defines a main() method. When the job file is imported, the main() method will be invoked, which would start the actual execution of the job file. All run calls should go inside this new main() method.

    import os
from ats.easypy import run

# process environment variables here
script_arg_a = os.environ['script_arg_a']

# all script runs goes into main()
def main():

    run('/path/to/my/script.py', script_arg_a = script_arg_a)

  • Tcl code can now be called via a Q magic function, making them appear like Python methods and objects. See Tcl documentation for details.

    from ats import tcl

tcl.q.info('exists', 'auto_path')

tcl.q.package('require', 'cAAs')

tcl.q.load_lib(file = '/path/to/file.tcl',
               os = 'NXOS',
               functional = 1)
