Features:

  • Modification to ‘devices’ attribute in the trigger, verification and PTS datafile. The devices are now passed as a list, and device_attributes is a dictionary for extra attributes. device_attributes is optional.

  • Fix a Cython issue.

devices: ['uut', 'helper']
devices_attributes:
    uut:
        exclude:
            - next_hello_time

  • orderer list can now be filter by using Regex!

order: ['connect', 'configure', '(profile_*)', 'configuration_snapshot']

  • Enhanced Robot Demos
