Upgrade Instructions

pip install --upgrade ats[full] # For internal user
pip install --upgrade pyats[full] # For DevNet user

If you have pyATS installed, you can use:

pyats version update

Warning

Backwards compatibility in genie.libs.parser:

ShowLispEthernetDatabase was removed as a duplicate of the ShowLispDatabaseSuperParser. The super parser output schema is similar but not fully backward compatible with the removed parser and may result in errors in tests that depend on specific values of the removed parser output.
