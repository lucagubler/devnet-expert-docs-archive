Definition of the pyATS Library code structure

The pyATS Library contains two types of packages:

  1. Core – The core and base classes, released as pip packages. We do not make the source code available.

  2. Feature libraries and components – The libraries that enable you to create and run reusable test cases. We make the source code available on GitHub for user-community contributions.
