A3.12 Dodatek C: Ukazi Git - Orodja za sisteme napeljave

Orodja za sisteme napeljave

V knjigi smo naleteli tudi na veliko število nižje nivojskih orodij sistema napeljave.

Prvo orodje, s katerim se srečamo, je ls-remote v Reference zahtevkov potega, ki ga uporabimo za ogled surovih referenc na strežniku.

ls-files smo uporabili v razdelkih Ročno ponovno združevanje datotek, Rerere in Indeks za ogled podrobnosti o vašem področju priprave.

V Reference vej smo omenili tudi rev-parse, ki omogoča pretvorbo skoraj katerega koli niza v objekt SHA-1.

Večina nizko nivojskih orodij sistema napeljave, ki smo jih obravnavali, pa je v Notranjost Gita, na kar je poglavje večinoma osredotočeno. Njihovi uporabi smo se poskušali izogniti skozi večino preostale knjige.
