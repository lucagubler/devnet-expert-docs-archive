6.6 GitHub - Qısa Məzmun

Qısa Məzmun

İndi bir GitHub istifadəçisisiniz. Bir hesab yaratmağı, bir təşkilatı necə idarə etməyi, depo’lar yaratmağı və push etməyi, başqalarının layihələrinə töhfə verməyi və başqalarının töhfələrini qəbul etməyi bilirsiniz. Növbəti fəsildə, sizi həqiqətən Git ustası halına gətirəcək mürəkkəb vəziyyətlərlə məşğul olmaq üçün daha güclü alətlər və tövsiyələr öyrənəcəksiniz.
