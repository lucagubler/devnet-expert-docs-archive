1.4 Иш бошланиши - Командалар сатри

Командалар сатри

Git ни ишлатишни турли хил йўллари бор. Git нинг асоси бўлмиш командалар сатри интерфейсидан ташқари кўпгина у ёки бу даражадаги имкониятларга эга фойдаланувчи график интерфейсидан ҳам фойдаланиш мумкин. Ушбу китобда биз Git ни командалар сатрида ишлатамиз. Командалар сатри бу барча командаларни ишга тушира олиш имконини берувчи ягона жой. Бундай имкониятга GUI лар эга бўлмайди ва улар Git нинг баъзи бир функционал ишларинигина бажара олишади холос. Агар сиз командалар сатрида бирор амални бажаришни билсангиз эҳтимол уни GUI да ҳам қандай қилинишини билиб олишингиз мумкиндир, бироқ буни тескариси ҳар доим ҳам ундай эмас. Албатта GUI ни танлашингиз ўзингизни табингизга боғлиқ, аммо билиб қўйиш керак Git командалар сатрида ишлаш имкони Git ни ўрнатишингиз билан пайдо бўлади.

Шунинг учун биз ўйлаймизки сиз Mac да терминални ва Windows да Powershell ни қандай очишни биласиз. Агар сизга биз бу ерда нималар ҳақида гапираётганимиз тушунарсиз бўлса у холда эҳтимол сиз китобда келтирилган мисолларни тушунишингиз учун бир мунча вақтга ушбу саволларни ўрганишга киришганингиз маъқулроқдир.
