1.4 Pierwsze kroki - Linia poleceń

Linia poleceń

Jest wiele sposobów użycia gita. Zarówno jako oryginalne narzędzia tekstowe jak i używając graficznych interfejsów z różnymi funkcjami. W tej książce będziemy korzystać z interfejsu lini komend. Po pierwsze, linia komend jest jedynym miejscem, w którym możesz używać wszystkich poleceń gita - większość interfejsów graficznych dla uproszczenia implementuje tylko pewien podzbiór funkcji. Jeśli wiesz jak użyć polecenia w linii komend to prawdopodobnie wiesz jak uruchomić je w trybie graficznym, w przeciwną stronę niekoniecznie jest to prawdą. Jeśli wybrałeś interfejs graficzny co jest kwestią gustu, wszyscy użytkownicy mają również zainstalowane narzędzia tekstowe gotowe do użycia.

Oczekujemy więc, że wiesz jak otworzyć Terminal w systemie Mac lub wiersz poleceń czy Powershell w systemie Windows.
