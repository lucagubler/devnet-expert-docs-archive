1.8 Ξεκινώντας με το Git - Ανακεφαλαίωση

Ανακεφαλαίωση

Σε αυτό το σημείο πρέπει να έχετε πλέον μία βασική γνώση του τι είναι το Git και πώς διαφέρει από ένα συγκεντρωτικό σύστημα ελέγχου έκδοσης που ενδεχομένως έχετε χρησιμοποιήσει στο παρελθόν. Επίσης, πρέπει να έχετε μια έκδοση του Git στο σύστημά σας η οποία είναι ρυθμισμένη με τα προσωπικά σας στοιχεία. Ήρθε πλέον η ώρα να μάθετε τα βασικά του Git.
