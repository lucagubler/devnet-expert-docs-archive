A1.5 Annexe A: Git dans d’autres environnements - Git dans Sublime Text

Git dans Sublime Text

Depuis la version 3.2, Sublime Text dispose de l’intégration git dans l’éditeur.

Ses caractéristiques sont :

  • La barre latérale indique le statut git des fichiers et répertoires avec un badge/icône.

  • Les fichiers et les répertoires correspondant au fichier .gitignore sont grisés dans la barre latérale.

  • Dans la barre de status, vous pouvez voir la branche git actuelle et le nombre de modifications en cours.

  • Toutes les modifications d’un fichier sont à présent visibles via une marque dans la gouttière.

  • Vous pouvez utiliser une partie des fonctions du client git Sublime Merge depuis Sublime Text. Cela requiert que Sublime Merge soit installé. Référez-vous à https://www.sublimemerge.com/.

La documentation officielle de Sublime Text se trouve sur https://www.sublimetext.com/docs/3/git_integration.html.
