名称

git-verify-tag - 检查标签的 GPG 签名

概述

git verify-tag [-v | --verbose] [--format=<格式>] [--raw] <标签>…​

描述

验证由 git tag 创建的 gpg 签名。

选项

--raw

将原始 gpg 状态输出打印到标准错误，而不是正常的人类可读输出。

-v
--verbose

在验证之前打印标记对象的内容。

<标签>…​

Git 标签对象的 SHA-1 标识符。
