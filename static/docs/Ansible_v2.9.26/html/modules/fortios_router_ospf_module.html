fortios_router_ospf – Configure OSPF in Fortinet’s FortiOS and FortiGate¶

New in version 2.8.

  • Synopsis

  • Requirements

  • Parameters

  • Notes

  • Examples

  • Return Values

  • Status

Synopsis¶

  • This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify router feature and ospf category. Examples include all parameters and values need to be adjusted to datasources before usage. Tested with FOS v6.0.5

Requirements¶

The below requirements are needed on the host that executes this module.

  • fortiosapi>=0.9.8

Notes¶

Note

  • Requires fortiosapi library developed by Fortinet

  • Run as a local_action in your playbook

Examples¶

- hosts: localhost
  vars:
   host: "192.168.122.40"
   username: "admin"
   password: ""
   vdom: "root"
   ssl_verify: "False"
  tasks:
  - name: Configure OSPF.
    fortios_router_ospf:
      host:  "{{ host }}"
      username: "{{ username }}"
      password: "{{ password }}"
      vdom:  "{{ vdom }}"
      https: "False"
      router_ospf:

        area:
